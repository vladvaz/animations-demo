Angular Animations

We can define animations in Angular setting the metadata animations: within the component decorator @Component().

There are several functions available:

We can define a trigger to start the animation. This trigger can be called inside the component controller.

Inside the trigger we can define the state where we place the rules for the CSS. We use the style function for that.

Next we define the transition, this is the animation rule between two states. Inside it, we put the animation function with the timings and the curves.
